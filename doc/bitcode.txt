OP	MNEMONIC	Description

0x00	H		Halt
0x01	A		Add operation (Signed)
0x02	S		Set to a fixed integer
0x03	M		Move Data Pointer (Forward and Reverse)
0x04	P		Set Pointer to a fixed integer
0x05	J		Jump unconditionally
0x06	JE		Jump if equal to zero
0x07	JZ		Jump if not zero
0x08	IO		IO
0x09	F		Fork
0x0A	D		Debug, print internal status
0x0B	X		Invaild Instruction, to indicate it's the end of code
